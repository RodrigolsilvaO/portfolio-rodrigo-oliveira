# Portfólio - Rodrigo Leite da Silva Oliveira

Portfólio pessoal profissional com dark mode, design responsivo e animações modernas.

## Sobre

Site de portfólio single-page construído do zero com HTML5, CSS3 e JavaScript vanilla (sem frameworks). Apresenta minha trajetória de 14+ anos como Técnico de Telecomunicações na Vivo Telefônica e minha transição para Desenvolvimento de Software.

## Tecnologias Utilizadas

- **HTML5** - Estrutura semântica e acessível
- **CSS3** - Variáveis CSS, Grid, Flexbox, animações
- **JavaScript** - IntersectionObserver, efeito de digitação, menu responsivo
- **Google Fonts** - Inter e JetBrains Mono

## Funcionalidades

- Dark mode como padrão
- Design responsivo (mobile-first)
- Menu hamburger no mobile
- Efeito de digitação no hero
- Animações de revelação ao rolar (IntersectionObserver)
- Barras de progresso animadas
- Timeline de experiência profissional
- Fundo com grid e partículas (CSS puro)
- Botão "voltar ao topo"
- Navegação com destaque da seção ativa
- Suporte a `prefers-reduced-motion` (acessibilidade)

## Seções

1. **Hero** - Apresentação com animação de digitação
2. **Sobre** - Resumo profissional e dados rápidos
3. **Habilidades** - Desenvolvimento e Telecomunicações
4. **Experiência** - Timeline com destaque para a Equipe V (VIP)
5. **Educação** - ADS (Estácio) e Gestão Pública
6. **Projetos** - Portfólio de projetos com links
7. **Contato** - E-mail, WhatsApp, localização e GitHub

## Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/RodrigolsilvaO/portfolio-rodrigo-oliveira.git
```

2. Abra o arquivo `index.html` no navegador.

Não é necessário instalar nenhuma dependência ou usar servidor local.
